# ac-dc-edit

A React-based dungeon editor built with Vite and TypeScript.

## Prerequisites

- Node.js (v18 or later)
- npm

## Clone

```bash
git clone https://github.com/philbgarner/ac-dc-edit.git
cd ac-dc-edit
```

## Install

```bash
npm install
```

## Run

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
